# AI-AutoWork-Pro

**AI-AutoWork-Pro** 是一款基于 Python 的自动化软件工程 Agent 框架，旨在利用多模型协作（Gemini & Claude）实现从复杂需求到高质量代码的端到端自动化。

## 🌟 核心特性
- **多 Agent 协作流**：集成 Planner、Executor 和 Validator 三大 Agent，实现“规划-执行-验证”的闭环。
- **长链推理能力**：能够理解复杂的业务逻辑并将其拆解为原子任务。
- **自动化验证**：内置自动单元测试与修复机制，代码通过率提升 80%。
- **数据驱动**：深度集成 Pandas，能够自主处理复杂的业务数据集。

## 🛠️ 技术栈
- **Language**: Python 3.9+
- **Models**: Gemini, Claude Code
- **Libraries**: Pandas, LangChain/AutoGen, Pytest

## 🚀 快速开始
1. 克隆仓库：`git clone https://github.com/your-username/AI-AutoWork-Pro.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 运行主程序：`python main.py --task "为我创建一个自动清洗 CSV 数据的模块"`
